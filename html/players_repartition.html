Données


Pour la réalisation de ce graphique nous avons utilisé les données de la liste des joueurs d'échecs enregistrés à la FIDE. Ces données sont disponibles en format XML ici. Elles donnent des informations telles que le nom, le sexe, le titre, la fédération, le fait que le joueur soit encore actif ou pas.
Afin de mettre les données dans le format necessaire pour la réalisation du graphique, nous avons écrit des scripts Python disponibles dans les sources du TP.


Visualisation


Pour illustrer la repartition des joueurs d'échecs, nous avons opté pour un Sunburst Zoomable avec 4 niveaux d'informations:
  1. Le titre: pouvant prendre les valeurs GM, WGM, IM, WIM, FM, WFM, CM et WCM,
  2. Le sexe: pouvant prendre les valeurs Homme et Femme,
  3. La fédération: représentant l'abbréviation à trois lettre la fédération nationale du joueur,
  4. L'activité: pouvant prendre les valeurs actif et inactif.

Variables visuelles


Nous avons utilisé deux variables visuelles dans ce graphique:
  • La Couleur, variable associative et sélective représentant la catégorie de données, suivant les attributs cités plus haut,
  • La Taille, variable ordonnée, associative, sélective et quantitave représentant l'éffectif de joueurs dans une catégorie.
